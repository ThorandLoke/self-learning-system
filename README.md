# Self-Learning System

A systematic self-learning framework for AI agents with structured review cycles, hands-on coding challenges, and progressive skill building.

## 🎯 What is this?

This Skill enables AI agents to learn continuously through:
- **Daily practice** - 5-stage learning routine
- **Weekly reviews** - Consolidate and plan
- **Hands-on challenges** - 4 difficulty levels (⭐ to ⭐⭐⭐⭐)
- **Progress tracking** - Monitor growth over time
- **Spaced repetition** - 1/3/7/14/30 day review cycles

## 🚀 Quick Start

```bash
# Install the skill
npx skills add ThorandLoke/self-learning-system

# Initialize your learning environment
python3 ~/.workbuddy/skills/self-learning-system/scripts/init-learning.py

# Start daily routine
python3 ~/.workbuddy/skills/self-learning-system/scripts/daily-routine.py
```

## 📚 Features

| Feature | Description |
|---------|-------------|
| 5-Stage Loop | Review → Challenge → Learn → Organize → Sync |
| 4 Difficulty Levels | ⭐(15min) ⭐⭐(1hr) ⭐⭐⭐(半天) ⭐⭐⭐⭐(全天) |
| Progress Tracking | Automatic stats and visualizations |
| Knowledge Base Integration | Syncs with memory system |
| Spaced Repetition | Smart review scheduling |
