# Learning Theory Reference

## Spaced Repetition

Review schedule for optimal retention:
- Day 1: Immediate review after learning
- Day 3: First spaced repetition
- Day 7: Weekly review
- Day 14: Bi-weekly review
- Day 30: Monthly review

## Active Recall

Better than re-reading:
1. Close notes
2. Try to remember key points
3. Check what you missed
4. Repeat until solid

## Deliberate Practice

Focus on weak areas:
- Identify gaps in knowledge
- Target specific skills
- Get immediate feedback
- Repeat with variations

## Project-Based Learning

Apply immediately:
- Build something real
- Solve actual problems
- Learn by doing
- Document insights
